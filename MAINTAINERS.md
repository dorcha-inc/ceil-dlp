# Maintainers

This document lists the maintainers of the ceil-dlp project. Maintainers are responsible for reviewing pull requests, managing releases, and guiding the project's direction.

## Current Maintainers List

- @jadidbourbaki

## Getting in Touch

Please file a [GitHub issue](https://github.com/dorcha-inc/ceil-dlp/issues) for general questions or feature requests.
Please see [SECURITY.md](SECURITY.md) for reporting security vulnerabilities.

## Becoming a Maintainer

If you're interested in becoming a maintainer, please reach out to the current maintainers through GitHub issues.

## Responsibilities

Maintainers are expected to:
- Review and merge pull requests
- Help triage issues
- Guide the project's technical direction
- Ensure code quality and test coverage
- Release new versions
- Foster a welcoming and inclusive community
